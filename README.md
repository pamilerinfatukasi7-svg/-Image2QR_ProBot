# 🎨 Image2QR Pro Bot

A powerful Telegram bot that converts text and images into customizable QR codes with style!

## ✨ Features

- **4 QR Styles**: Default, Rounded, Circle, Gradient
- **11 Colors**: Black, Red, Blue, Green, Purple, Orange, Pink, Cyan, Teal, Gold, Magenta
- **Logo Embedding**: Upload images to embed as logos
- **High Quality**: Professional grade QR codes
- **User Preferences**: Settings saved per user

## 📋 Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/help` | Help guide |
| `/settings` | View preferences |
| `/reset` | Reset to defaults |

## 🚀 Deployment

1. Create bot via @BotFather
2. Fork this repository
3. Deploy on Railway
4. Add `TELEGRAM_TOKEN` variable

## 🛠️ Tech Stack

- Python 3.12
- python-telegram-bot 21.5
- qrcode 7.4.2
- Pillow 10.1.0
- Railway
